Installation

Copy the content of the deploy folder to your server or local server, for example, to the folder "pdftoimg".

Usage

  1. Open the URL of your application in the browser. For example, if your domain is example.com and you copied it to the folder pdftoimg, open example.com/pdftoimg
  2. Adjust the settings for the conversion if needed.
  3. Upload the PDF. The conversion will automatically start.
  4. Once the conversion is completed, download the zip file by clicking the Download button.

Settings

Image height
The height of the generated image in pixels, recommended between 1000 and 3000.
Thumbnail height
The height of the generated thumbnail image in pixels, recommended between 100 and 300.
Image quality
Between 0 and 1. Recommended 0.8. A higher value does not give noticeable quality improvements but increases the image size.
Image type
Choose between JPEG and WebP. Recommended is WebP, which is smaller in size at the same quality and is supported in all modern browsers.

Real3D Flipbook

Real3D Flipbook jQuery Plugin is used to create realistic 3D flipbooks from PDF or images inside your website.

It is sold separately here.

With PDF to Image Converter, you can create a set of images and JSON from a PDF that can be used to create a flipbook with working PDF links, text selection, and text search.

For example, you can convert the PDF file book1.pdf and get a folder book1 with an image, thumbnail image, and JSON for each PDF page.

You can use those files to create a flipbook by passing an array of pages that have src, thumb, and json, like this: 

jQuery("#container").flipBook({
  pages: [
    { src: "book1/1.webp", thumb: "book1/thumb1.webp", json: "book1/1.json" },
    { src: "book1/2.webp", thumb: "book1/thumb2.webp", json: "book1/2.json" },
    ...
  ]
});
